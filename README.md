# SwapSkill

A Flutter application designed to connect people who want to exchange skills, learn from each other, and grow through peer-to-peer collaboration.

## Overview

SwapSkill enables users to create profiles, list their skills, discover others with complementary skills, and initiate skill exchange sessions. The goal is to build a community where learning is accessible without traditional barriers.

## Features

* User authentication and profile setup
* Skill listing and categorization
* Skill matching between users
* Search and discovery of users by skills
* Chat or connection requests
* Session scheduling for skill exchange
* Dark and light theme support
* Responsive UI for mobile devices

## Tech Stack

* Flutter for cross-platform mobile development
* Dart as the programming language
* State management approach depending on implementation
* Material Design components
* Firebase

## Getting Started

Follow these steps to run the project locally

* Install Flutter SDK
* Run flutter doctor to verify setup
* Clone the repository
* Run flutter pub get
* Launch the app using flutter run
